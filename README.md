# SSI LOG – Nhật Ký Giao Dịch AI

Ứng dụng nhật ký giao dịch chứng khoán Việt Nam tích hợp trí tuệ nhân tạo Gemini để phân tích tâm lý và chiến lược đầu tư.

## Tính năng
- 📊 Quản lý giao dịch: Thêm, sửa, xóa và theo dõi danh mục.
- 🤖 Phân tích AI: Kết nối trực tiếp với Gemini API để nhận định về tâm lý và rủi ro.
- 💾 Dữ liệu: Lưu trữ offline trên trình duyệt, hỗ trợ xuất/nhập file JSON và CSV.
- 🌓 Giao diện: Hỗ trợ chế độ Sáng/Tối.

## Cách sử dụng
1. Truy cập ứng dụng.
2. Cấu hình Gemini API Key trong phần Cài đặt.
3. Bắt đầu ghi nhật ký giao dịch của bạn.
